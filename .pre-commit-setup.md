# Pre-commit Hook Setup

## Tự động chạy khi commit:
- ✅ `ruff format` - Auto-fix formatting
- ✅ `ruff check --fix` - Auto-fix lint issues
- ✅ `pytest` - Quick test run

## Cách dùng:

### Commit bình thường (tự động chạy hooks):
```bash
git add .
git commit -m "your message"
# → Tự động format + fix lint + run tests
```

### Skip hooks (khi cần):
```bash
git commit -m "wip" --no-verify
```

### Chạy thủ công trên tất cả files:
```bash
pre-commit run --all-files
```

### Uninstall (nếu không muốn dùng nữa):
```bash
pre-commit uninstall
```

## Lưu ý:
- Hooks chỉ chạy trên files bạn đang commit (staged)
- Nếu có lỗi không auto-fix được, commit sẽ bị block
- Tests chạy nhanh với `-x` (stop at first failure)
